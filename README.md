# riess.no

Personal website of Mike Riess - Senior Research Scientist at Telenor Research, working on applied AI, business analytics and process mining.

Live at **https://mikeriess.github.io/**

## Stack

Built with [Jekyll](https://jekyllrb.com/) and deployed via GitHub Pages. The design is a custom warm dark theme (single stylesheet on CSS custom properties, self-hosted Source Serif 4 for headings, vendored KaTeX for math). The layout structure was originally based on [Contrast](https://github.com/niklasbuschmann/contrast) by Niklas Buschmann.

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000.
